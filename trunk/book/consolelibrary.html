 ConsoleLibrary

This is the start of a library to make developing console-based applications easier.

needs console

  • console_init
    • -- n
    • Initialize the console subsystem to key breaking mode
  • console_cleanup
    • -- n
    • Reset the console subsystem to the OS default
  • getkey
    • -- n
    • Read a key and return a single value code
  • getchar
    • -- n
    • Read a single byte from the keyboard; specials keys may return multiple byte codes.
  • console_dimensions
    • -- x y
    • Return the rows and columns of the console window
  • xy
    • x y --
    • Set the cursor to x,y
  • home
    • --
    • Reset the cursor to the top left corner
