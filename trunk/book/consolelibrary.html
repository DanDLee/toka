 ConsoleLibrary

This is a library to make developing console-based applications easier.

Provides:

  • xy
    • Position the cursor to any position on the screen
    • x y --
  • home
    • Position the cursor to the top left corner of the screen
    • --
  • console_init
    • Initialize the console to direct keyboard input
  • console_cleanup
    • Reset console to previous input model
  • getkey
    • Read a single keypress. Use console_init first.
  • getchar
    • Read a single keypress. Use console_init first.
  • console_dimensions
    • Return the height and width of a console
