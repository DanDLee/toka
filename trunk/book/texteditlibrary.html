 TextEditLibrary

Toka has a hybrid library that provides a Pico/Nano style text editor. This is based on a simple text editor originally developed by Tom Novelli.

needs textedit

  • edit.load
    • $file --
    • Load a file ($file = filename) into the editor buffer
  • edit.save
    • $file --
    • Save the editor buffer to a file ($file = filename)
  • edit.start
    • --
    • Begin editing the file in the editor buffer
  • edit.redraw
    • --
    • Redraw the editor screen
