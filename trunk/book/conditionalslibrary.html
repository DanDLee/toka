 ConditionalsLibrary

This library provides additional conditionals to help with comparisions.

Provides:

  • >=
    • Return TRUE if x greater than or equal to y. FALSE otherwise.
    • x y -- flag
  • <=
    • Return TRUE if x less than or equal to y. FALSE otherwise.
    • x y -- flag
  • >!=
    • Return TRUE if x greater than or not equal to y. FALSE otherwise.
    • x y -- flag
  • <!=
    • Return TRUE if x less than or not equal to y. FALSE otherwise.
    • x y -- flag
