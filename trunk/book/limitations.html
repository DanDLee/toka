Limitations

  • Cell size is machine dependent, generally 32 bits (4 bytes) or 64 bits (8 bytes) in size. cell-size returns the actual size in bytes.
  • Character size is machine and OS dependent. Normally 8 bits (1 byte). char-size returns the actual size in bytes.
  • Quotes can be up to 32 elements (values, primitives, embedded quotes, calls to other quotes, string pointers, etc) in length.
  • The data stack can hold up to 100 cells.
  • The return stack can hold up to 1024 cells.
