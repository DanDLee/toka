 StructLibrary

This library allows for creation of variable sets with a specific form.

Provides:

  • struct{
    • Begin a structure
    • --
  • }struct
    • End a structure
    • "name" --
  • .char
    • Create a char-size variable
    • string --
  • .cell
    • Create a cell-size variable
    • string --

Example: 

  needs struct

  struct{
   " x" .cell
   " y" .cell
  }struct POINT

  POINT foo

  foo.x @ .
  foo.y @ .
