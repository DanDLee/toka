 LocalsLibrary

This library adds a form of local variable to Toka. Words using this are not reentrant, but it may help make some pieces of code cleaner.

Provided:

  • local[
    • Start a new localized quote
    • "names.... |" -- pointer
  • ]local
    • Close a localized quote
    • --

Example: 

 local[ a b c | 
        a ! b ! c !
        a @ c @ * b @ + .
 ]local is foo
 foo
