 CaseLibrary

This library provides basic functionality for a switch/case style construct for both numbers and strings.

Provides:

  • <cond>
    • Data element, stores the value to check against
  • switch
    • Stores the value to check against into the <cond> variable
    • number --
  • case
    • Compare NOS to value in <cond> and invoke the quote (TOS) if they match
    • number quote --
  • string-case
    • Compare string in NOS to value in <cond> and invoke the quote (TOS) if they match
    • string_pointer quote --

Example: 

 needs case
 
 [ switch
   1 [ ." One!\n"   ] case
   2 [ ." Two!\n"   ] case
   3 [ ." Three!\n" ] case
   4 [ ." Four!\n"  ] case
   5 [ ." Five!\n"  ] case
 ] is try
 
 ." Inputs are: 1 3 5 2 5 4\n"
 1 try
 3 try
 5 try
 2 try
 5 try
 4 try
