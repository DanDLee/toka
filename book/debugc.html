 debug.c

Use

This provides a very small collection of simple tools that can be helpful when trying to track down bugs. It is not intended to be a full-scale debugging tool.

Functions Provided

words()
Display a list of all named words

display_stack()
Display all items on the stack.

gc_info()
Display information about Toka's memory use

Primitives Provided

:words
:stack
:gc
