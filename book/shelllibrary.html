 ShellLibrary

This library provides the fundamental support for calling external files and doing actions normally done at the shell.

The provided API follows: 

  remove
  ( $-n )
  Delete file '$'

  system
  ( $-n )
  Run string '$' as a shell command

  getpid
  ( -n )
  Return the current process ID

  fork
  ( -f )
  Fork the current process.
 
  basename
  ( $-$ )
  Return the base name (name+extension), of a file.
  Strips any directories from the filename.

  mkdir
  ( $m-n )
  Create a directory named "$" with a mode of "m".

  fnmatch
  ( ab-f )
  Returns TRUE if filename "a" matches the pattern in "b". 
  Returns FALSE otherwise.
