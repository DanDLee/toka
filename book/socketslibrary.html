 SocketsLibrary

This library was provided by erider to allow working with sockets. It is sufficient to handle the needs of a basic IRC client, HTTP server, and other tasks.

Requires:

  • sockets.so (built during installation)

Provides:

  • pBind
    • n -- x
    • Bind to a specific port
  • pConnect
    • $ n -- s
    • Connect to host (string '$') at port 'n'. Returns a socket.
  • pAccept
    • n -- x
  • pRead
    • s b l -- n
    • Attempt to read 'l' chars from socket 's' into buffer 'b'. Returns the number of bytes read.
  • pWrite
    • s b l -- n
    • Attempt to write 'l' bytes from buffer 'b' to socket 's'. Returns the number of bytes written.
  • pClose
    • s -- x
    • Attempt to close socket 's'.
