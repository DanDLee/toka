 First Steps

At present, Toka provides a basic listener as the only interface. This is a console-based enviroment where you can type commands and get instant feedback. Let's get started: 

  crc@strider: toka

If you built and installed Toka properly, it should start, displaying a message like: 

  toka (svn build)

At this point, you can begin entering code. Let's start by exploring the enviroment a little: 

  :names

Invoking :names will display all named elements in the currently running Toka. Don't panic at the number of items -- you only need to learn a handful of them to write code in Toka.
