 Sandbox

This is place to experiment with Notebook's way of marking up text. Edit this page to see how you type things; then save it to see what it looks like when you're browsing.

You Can Have Section Headers

  • You can have bulleted lists.
  • You can type bold, italic, and monospace text, or any combination.
  • You can have inline header text.
  • You can have very small text.
  • You can strike things out.
    • You can simply indent a paragraph without adding a bullet.

Paragraphs end with the first blank line, bullet item, or indented line. No matter how many lines long a paragraph is, Notebook will wrap it for display so that it looks nice. 

 If you leave a space character at the beginning of a paragraph, the whole
 paragraph is "Preformatted".  That means that it's displayed in a monospace
 font, exactly as is.  It's conventional to put whitespace at the beginning of
 every line of a preformatted paragraph, but that's not necessary.

Preformatted text is useful for typing in tables and things like that.
