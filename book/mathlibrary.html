MathLibrary

 Provides:
     abs
     ( n-n )
     Returns the absolute value of 'n'

     random
     ( -n )
     Returns a random number

     srandom
     ( s-x )
     Set a new seed for the random number generator

     ^
     ( xy-z )
     Returns x^y power

     sqrt
     ( x-y )
     Returns the square root of x

     sqrt-closer
     Used internally by sqrt
